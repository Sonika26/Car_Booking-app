# 🚗 Rentroo - Car Booking App

Rentroo is a modern **Car Booking Web Application** built with **React** that enables users to search, book, and manage rental cars through an intuitive and responsive interface. The application uses **Firebase Authentication** for secure user login and **Cloud Firestore** to store user information and booking history.

## ✨ Features

- 🔐 Secure user authentication with Firebase Authentication
- 🚗 Search and explore rental cars by destination
- 📋 Browse available vehicles with detailed information
- 📅 Book rental cars with a simple booking process
- 📖 View current and previous bookings in the **My Bookings** page
- ☁️ Store user profiles and booking history using Cloud Firestore
- 📱 Responsive and user-friendly interface

---

## 📄 Pages

- **Home** – Search and explore rental cars by destination.
- **Listings** – Browse available rental cars.
- **My Bookings** – View booking history and active bookings.
- **Contact** – Contact and support page.
- **Login / Signup** – Secure user authentication.

## 🚀 Usage

1. Create an account or log in.
2. Search for rental cars by entering your destination.
3. Browse available vehicles and book your preferred car.
4. View and manage your bookings from the **My Bookings** page.
